# 3D Chess Game

A fully functional 3D Chess game built using HTML, CSS, JavaScript, and Three.js.

## Live Demo
Check out the live demo here: [https://chess-3d-game.vercel.app/](https://chess-3d-game.vercel.app/)

## Features
*   Interactive 3D board and pieces using `Three.js`
*   Custom 3D GLTF/GLB models for chess pieces
*   Valid move highlighting and piece selection
*   Fully responsive design
*   Clean UI with HTML and vanilla CSS

## Technologies Used
*   HTML5
*   CSS3
*   JavaScript (ES6)
*   Three.js (for 3D rendering)

## Local Setup
1. Clone the repository.
2. Serve the directory using a local web server (e.g., Live Server extension in VS Code, or `python -m http.server`). *Note: Due to ES modules and CORS policies, simply double-clicking `index.html` may not load 3D assets correctly.*
3. Open the local server URL in your browser to play.
